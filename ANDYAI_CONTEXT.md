# ANDYAI_CONTEXT.md

## Identity

This repository defines AndyAI Second Brain: a governed work-memory layer for humans, AI agents, projects, repositories, decisions, evidence and reusable skills.

## Current Focus

The system is being bootstrapped as a canonical AndyAI pillar connected to:

- TAP-TAP v2
- Memory Runtime Lab
- Knowledge Factory
- Canonizer
- Director
- Human Attention
- Agent Discipline Standard

## Operating Rule

Do not load all memory at once.
Start from this root context.
Then open only the relevant project, skill, evidence or decision file.

## Core Formula

RAG finds fragments.
Second Brain preserves work continuity.

## Serbian Formula

## Permission Boundary

AI may organize, suggest, draft, inspect and verify.
Human approval is required for publishing, deleting canon, changing production, sending client materials or executing high-impact actions.
