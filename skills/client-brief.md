# Skill: client-brief

## Purpose

Prepare a client-ready explanation from Second Brain memory.

## Inputs

- client name
- topic
- language
- desired format
- evidence links

## Steps

1. Identify audience level.
2. Select only relevant context.
3. Translate if required.
4. Produce clear explanation.
5. Mark draft as needing human approval before sending.

## Output Format

Client Brief Draft.
