# Skill — Governed Agent Execution

## Purpose
Prepare agent execution safely with approval and evidence.

## Steps
1. Read mission context.
2. Create execution plan.
3. Identify permission level.
4. Create human approval checklist.
5. Run dry-run first.
6. Capture evidence.
7. Inspect result.
8. Stop if any condition fails.
9. Record rollback note.
10. Produce execution report.

## Never do
- bypass approval
- delete canon silently
- run destructive actions without human authorization
