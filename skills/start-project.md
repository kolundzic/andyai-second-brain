# Skill: start-project

## Purpose

Create a structured project context for a new AndyAI project.

## Inputs

- project name
- project purpose
- source signal or user request
- connected repos
- first milestone

## Steps

1. Identify the project purpose.
2. Classify the project as Project, Area, Resource, or Archive.
3. Create project context.
4. Define first milestone.
5. Define evidence requirements.
6. Define human approval boundaries.

## Output Format

`PROJECT_CONTEXT.md`
