# Skill: repo-status

## Purpose

Summarize the current state of a repository.

## Inputs

- repo name
- latest known commit
- latest known tag
- verification status
- deployment status

## Steps

1. Identify latest milestone.
2. Identify proof of completion.
3. Identify pending action.
4. Identify risks.
5. Produce concise status report.

## Output Format

Repo Status Report.
