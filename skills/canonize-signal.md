# Skill: canonize-signal

## Purpose

Turn a public or private signal into AndyAI canon.

## Inputs

- URL, screenshot, PDF, note, or user message
- project context
- current AndyAI ecosystem map

## Steps

1. Identify core idea.
2. Separate useful truth from hype.
3. Map to AndyAI projects.
4. Extract risks.
5. Extract practical consequences.
6. Propose repo or product consequence.
7. Save canon block.

## Output Format

Canonical Markdown block.
