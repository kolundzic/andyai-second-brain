# Governance Model

## Human Authority

AI may:

- organize
- suggest
- draft
- classify
- inspect
- verify
- summarize

AI may not, without explicit human approval:

- publish public material
- delete canon
- change production
- send client material
- execute high-impact actions
- modify money, health, legal, or identity-critical decisions

## Status Values

- draft
- suggested
- needs_review
- approved
- rejected
- locked
- archived

## Evidence Rule

Important memory entries must include source, reason, confidence, and evidence link.
