# PARA Workspace Layer

PARA means:

- **Projects** — active outcomes with a finish line
- **Areas** — ongoing responsibilities without a final deadline
- **Resources** — useful reference material
- **Archives** — completed, frozen or inactive material

## AndyAI Meaning

Projects are active udarci.
Areas are long-term pillars.
Resources are knowledge and signals.
Archives are locked history.

## Simple Rule

If it has a next action, it is probably a Project.
If it must be maintained, it is probably an Area.
If it teaches or supports, it is probably a Resource.
If it is finished or inactive, it is probably an Archive.
