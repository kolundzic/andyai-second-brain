# Roadmap

## v0.1.0 — Second Brain Architecture Bootstrap

Repository foundation, context file, canon, docs, folders, skills, schemas, examples, verifier.

## v0.2.0 — PARA Workspace Layer

Projects, Areas, Resources, Archives, Evidence, Reports, Skills, Decisions.

## v0.3.0 — Project Context Standard

Every project receives a structured `PROJECT_CONTEXT.md`.

## v0.4.0 — Skill-as-Markdown Standard

Reusable Markdown workflows for canonization, repo status, meeting debrief, client brief, and TAP-TAP summaries.

## v0.5.0 — Evidence-Backed Memory

Every important memory entry gets source, claim, evidence, confidence, status, and connected project.

## v0.6.0 — Decision Log

Decision records preserve what was decided, when, why, and what comes next.

## v0.7.0 — Human Approval Gate

Clear boundary between AI suggestion and human-approved action.

## v0.8.0 — Repo Memory Bridge

Track repo tags, commits, packs, verification, deployment, and next strikes.

## v0.9.0 — Client Memory Bridge

Track client context, meetings, promises, materials, language, and follow-up.

## v1.0.0 — AndyAI Second Brain MVP Lock

Personal, project, repo, signal, evidence, skill, decision, approval, and report memory in one governed system.
