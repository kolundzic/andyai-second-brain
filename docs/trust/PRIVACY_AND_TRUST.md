# Privacy and Trust

## Core principle

Trust is not a feature. Trust is the product boundary.

## Why privacy matters

AI output can accidentally contain sensitive data, internal notes, client details, emails, tokens or raw evidence.

AndyAI Second Brain treats public output as something that must be checked.

## Safety layers

- privacy firewall
- public output scan
- redaction rules
- audit reports
- human approval
- GitHub leak guard
- Vercel public export gate
- Supabase RLS blueprint

## Canonical rule

Machine output is not automatically safe.

Safe output is machine output that passed privacy, trust and human approval gates.
