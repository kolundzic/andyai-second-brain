# Supabase Local Setup Checklist

1. Create Supabase project.
2. Confirm project URL.
3. Add anon key to local env.
4. Keep service role key server-only.
5. Review migration SQL.
6. Apply schema in a test project first.
7. Enable RLS.
8. Test tenant isolation.
9. Test storage bucket privacy.
10. Run verifier before real client data.
