# Operator Registry Standard

Operators are guided service workflows.

Examples:
- GitHub Operator
- Vercel Operator
- Supabase Operator
- Stripe Operator
- DNS Doctor
- Recovery Doctor

Operators must expose:
- purpose
- required inputs
- risk level
- human approval boundaries
- safe checks
- evidence outputs
- stop conditions
- rollback notes

Canon:
Operators do not remove human control. They remove confusion around repetitive service setup.
