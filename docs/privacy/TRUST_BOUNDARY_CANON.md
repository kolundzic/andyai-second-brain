# AndyAI Trust Boundary Canon

## Core principle

Trust is not a feature. Trust is the product boundary.

## Rules

No client data in GitHub.  
No secrets in GitHub.  
No raw evidence in public.  
No public report without human approval.  
No Supabase table without RLS.  
No deployment without privacy gate.

## Canon line

Machine output is not automatically safe.  
Safe output is machine output that passed privacy, trust and human approval gates.
