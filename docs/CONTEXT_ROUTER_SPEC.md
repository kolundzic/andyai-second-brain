# Context Router Specification

The context router decides which file to open next.

## Routing Questions

1. Is this about a project?
2. Is this about an area/responsibility?
3. Is this a resource/reference?
4. Is this archived history?
5. Is this evidence?
6. Is this a decision?
7. Is this a skill/action request?

## Rule

Do not load everything. Load the smallest useful context.
