# Brain Dialog Spec

The Brain Dialog is the primary future interaction window between the user and AndyAI Second Brain.

The user should be able to:

- ask questions
- upload documents
- ask the Brain to remember information
- search by meaning, time, event, client or topic
- sort and compare records
- generate reports
- prepare printable output
- prepare public or client-facing pages
- keep sensitive material behind privacy, trust, logic-layer and human approval gates
