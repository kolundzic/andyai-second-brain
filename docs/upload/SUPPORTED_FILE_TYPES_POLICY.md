# Supported File Types Policy

Initial allowed candidates:

- PDF
- Word: DOC/DOCX
- Excel: XLS/XLSX/CSV
- TXT/Markdown
- PNG/JPG/JPEG/WEBP screenshots and scans

Blocked by default:

- executables
- scripts
- archives with unknown contents
- password-protected files
- files above configured size limit
- files that fail privacy or logic-layer gates
