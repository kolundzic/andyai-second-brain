# Document Upload Processing Spec

Supported input candidates:

- PDF
- Word / DOC / DOCX
- Excel / XLS / XLSX
- CSV
- TXT / Markdown
- PNG / JPG screenshots
- camera scans

Processing stages:

1. intake
2. file type detection
3. metadata extraction
4. safety scan
5. text extraction / OCR placeholder
6. classification
7. memory bucket assignment
8. summary generation
9. report or action preparation
