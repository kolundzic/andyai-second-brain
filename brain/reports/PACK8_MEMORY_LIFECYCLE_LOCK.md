# PACK8 — Memory Lifecycle Lock

AndyAI Second Brain now has a governed memory lifecycle layer:

- inbox
- active
- canon
- archive
- stale review
- cleanup request
- delete approval
- retention policy
- memory health
- archive index
- lifecycle audit bridge
- lifecycle export

Canon: memory must have a lifecycle. Not every note becomes canon. Not every stale item is deleted. Human approval governs destructive cleanup.
